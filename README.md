## âš¡ Quick Start

### Prerequisites

- **Bun** (Lightning-fast runtime) - `npm install -g bun`
- **Node.js** 18+ 
- **PowerShell** (Windows) or **Bash** (Unix)
- **Git** with safe directory configuration

### 30-Second Setup

```bash
# Clone and navigate
git clone https://github.com/371-Minds/os.git
cd os

# Lightning-fast dependency installation with Bun
bun install

# Verify system readiness
bun run quick-status

# Start the cognitive interface
bun run start:cognitive-interface
```

### First-Time Setup

```bash
# Configure Git safe directory
git config --global --add safe.directory f:/os-main

# Install dependencies with Bun optimization
powershell -ExecutionPolicy Bypass -File project-management/AB/scripts/bun-install.ps1

# Verify Nx workspace
bun nx --version
bun nx show projects

# Start development environment
bun run start:dev
```
