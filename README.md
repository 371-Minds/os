### Prerequisites

- Node.js 18+
- npm or yarn  
- Git
- ElizaOS runtime

### Installation & Setup

```bash
# Clone and setup
git clone <repo-url>
cd 371-minds-os
npm install

# Validate Nx workspace
npx nx graph

# Install the revolutionary plugin
cd packages/elizaos-plugins/nx-workspace
npm install

# Run the workspace analysis
npx nx run elizaos-plugin-nx-workspace:test
```

```bash

npm run graph

# Build everything efficiently  
npm run build
```
